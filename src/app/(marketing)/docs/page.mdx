# Odyhook docs

Odyhook is a self-hosted webhook router — it receives webhooks from providers (Stripe, GitHub, etc.), verifies signatures, persists every event, optionally filters and transforms, and reliably forwards to downstream destinations with retries.

**Start here:** [Quickstart](/docs/quickstart).
